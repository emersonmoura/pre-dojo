O problema não especifica nenhuma linguagem/plataforma, portanto tomei a liberdade de fazer utilizando Ruby e Rails

Para ver o projeto em execução basta acessar o seguinte link: http://ranking-amil.herokuapp.com/

A versão atual não contempla multiplas partidas. 

Testes unitários ficam na pasta spec/models

Classes ficam na pasta app/models - app/controllers

Telas em app/views

Para executar o projeto localmente, instale ruby e o rails: http://rvm.io/rvm/install

Para executar os teste digite na raiz do projeto o comando: rspec 

Para "subir" o serviço digite na raiz do projeto o comando: rails server (endereço para acesso: localhost:3000)
